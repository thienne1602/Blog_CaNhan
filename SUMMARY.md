## 🚀 BƯỚC TIẾP THEO: DEPLOY

### Option 1: GitHub Actions (Khuyến nghị - Tự động) ⭐

```bash
# 1. Push code lên GitHub
git add .
git commit -m "feat: Hoàn thành đồ án SSG Blog"
git push origin main

# 2. Cấu hình GitHub Pages
# - Settings > Pages > Source: GitHub Actions
# - Settings > Actions > Workflow permissions: Read and write

# 3. Đợi 2-5 phút → Website live!
```

### Option 2: Deploy thủ công

```powershell
# Build
npm run build

# Deploy bằng gh-pages
npm run deploy
```
